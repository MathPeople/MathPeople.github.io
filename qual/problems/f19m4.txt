<problem><p>Let \(g\) be an entire function and define \(f(z)\) by</p><p>\[f(z):=\frac{g(z)}{\sin(-iz^2)}.\]</p><p>Here, we add the assumption that \(g(z)\neq0\) for any zero of \(\sin(-iz^2)\), and we define \(f\) only where \(\sin(-iz^2)\neq0\).</p><p>(a) What are the zeros of \(\sin(-iz^2)\)? Sketch their locations in \(\mathbb{C}\).</p><p>(b) For \(n\in\mathbb{N}\), let \(a=\sqrt{2n\pi}\) lie on the positive real axis. Determine the radius of convergence of the power series for \(f\) expanded about \(a\).</p></problem><solution><p>a) Recall \(\sin(z)=\frac{1}{2i}(e^{iz}-e^{-iz})\), this is the most computationally-convenient definition of complex \(\sin\). This vanishes precisely when \(e^{iz}=e^{-iz}\), which is equivalent to \(e^{2iz}=1\). If \(z=x+iy\) then this is also</p><p>\[e^{-2y}e^{2ix}=1\]</p><p>Taking modulus implies \(e^{-2y}=1\), so this is only satisfied when \(y=0\). Assume \(y=0\), we are solving \(e^{2ix}=1\). We know that \(e^0=1\) and we know \(e^z\) is \(2\pi i\)-periodic in \(z\), so this is satisfied precisely when \(x=\pi k\) for some integer \(k\). Thus \(\sin(z)=0\) if and only if \(z=\pi k\) for some integer \(k\).</p><p>Substituting \(w=-iz^2\) and using our knowledge that \(\sin(w)=0\) if and only if \(w=2\pi k\) gives that \(\sin(-iz^2)=0\) if and only if \(-iz^2=2\pi k\), or \(z=\pm\sqrt{2\pi ik}\) for \(k\in\mathbb{Z}\).</p><p>b) The radius of convergence will be the largest it can be without surpassing a pole. The closest pole to \(a\) is \(0\), so the radius of convergence is \(a\) itself.</p></solution><topics><ExpAndTrig/><EntireFunction/><RadiusOfConvergence/><Meromorphic/></topics><instructors><d/></instructors>
