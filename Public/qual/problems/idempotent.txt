<problem><p>Let \(f\) be a holomorphic function on the region \(\Omega\) such that \(f(\Omega)\cap\Omega\neq\emptyset\) and \(f(f(z))=f(z)\). Prove \(f\) is either constant or the identity.</p></problem><solution><p>Taking the derivative gives</p><p>\[f&apos;(z)=f&apos;(f(z))f&apos;(z)\]</p><p>If \(f\) is not constant then</p><p>\[1=f&apos;(f(z))\]</p><p>Again</p><p>\[0=f&apos;&apos;(f(z))f&apos;(z)\]</p><p>Remember \(f\) is not constant, so this means \(f&apos;&apos;(f(z))\) constantly vanishes. Moreover, \(f(\Omega)\) is open by the open mapping theorem, so \(f&apos;&apos;\) vanishes on an open set. The uniqueness principle states that \(f&apos;&apos;\) must then be constantly 0, so there are constants \(a,b\) with</p><p>\[f(z)=az+b\]</p><p>Solving the initial requirement is</p><p>\[az+b=a(az+b)+b=a^2z+ab+b\]</p><p>\[a^2z-az+ab=0\]</p><p>\[a(az-z+b)=0\]</p><p>If \(a\) were 0, \(f\) would be constant. Thus we have assumed \(a\neq0\). Then \(az-z+b=0\) for all \(z\). That is,</p><p>\[z(a-1)=-b\]</p><p>If \(a\neq1\) then this would state that the only complex number is \(\frac{-b}{a-1}\), a clear falsehood. Thus \(a\) must be 1. This forces \(b\) to be 0 and \(f\) to be the identity.</p></solution><topics><OpenMappingTheorem/><UniquenessPrinciple/><ElementaryDefinitions/></topics><instructors><e/></instructors>
