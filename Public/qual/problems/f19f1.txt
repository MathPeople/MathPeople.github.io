<problem><p>State the following, clearly and completely:</p><p>(a) What it means for a complex-valued function to have an isolated singularity.</p><p>(b) What it means for a complex-valued function to have a pole.</p><p>(c) Some version of Cauchy&apos;s integral formula giving the value of \(f^{(k)}\) for an analytic function \(f\).</p></problem><solution><p>a) Suppose \(\Omega\) is a region in \(\mathbb{C}\) and that there is some \(a\in\mathbb{C}, r&gt;0\) such that \(B_r(a)\cap\Omega=B_r(a)\setminus\{a\}\). Then we say that a holomorphic function \(f:\Omega\to\mathbb{C}\) has an isolated singularity at \(a\).</p><p>b) Suppose \(\Omega\) is a region and \(f:\Omega\to\mathbb{C}\) has an isolated singularity at \(a\in\mathbb{C}\). We say that \(f\) has a pole at \(a\) if \(\lim_{z\to a}|f(z)|=\infty\).</p><p>c) Let \(f:\Omega\to\mathbb{C}\) be holomorphic and let \(a\in\Omega, r&gt;0\) be such that \(\overline{B_r(a)}\subset\Omega\). Then for all integers \(k\geq0\),</p><p>\[\frac{2\pi i}{k!}f^{(k)}(a)=\int_{|z-a|=r}\frac{f(z)}{(z-a)^{k+1}}dz\]</p><p> </p></solution><topics><CauchyIntegralFormula/><PointSetTopology/><ClassificationOfSingularities/></topics><instructors><d/></instructors>
